# World Countries Data Analysis

This mini-project performs Exploratory Data Analysis on a dataset of global countries. It utilizes Python and pandas to clean, process, and analyze macroeconomic and demographic metrics to uncover regional and global trends.

## Technologies Used

* **Python 3**
* **pandas**
* **numpy**
* **matplotlib** * **seaborn** ---

## Core Operations

* **Data Cleaning:** Handling missing values and ensuring correct data types for numerical analysis.
* **Exploratory Data Analysis (EDA):** Generating descriptive statistics for key metrics across different nations.
* **Regional Aggregation:** Grouping countries by continent or region to compare aggregated economic and demographic performance.
* **Data Visualization:** Creating charts and graphs to illustrate distributions and correlations between variables.

---

## Dataset Structure

The analysis is based on the `Countries.csv` dataset. Key features typically include:

| Column | Description |
| :--- | :--- |
| **Country** | The name of the nation. |
| **Region** | The geographical or geopolitical region. |
| **Population** | Total population count. |
| **Area** | Total land area in square miles or kilometers. |
| **GDP** | Gross Domestic Product per capita. |
| **Infant Mortality** | Infant mortality rate per 1,000 live births. |

---

## Setup and Execution

1. Clone this repository to your local machine.
2. Install the required dependencies via your terminal: `pip install pandas numpy matplotlib seaborn`
3. Ensure the `Countries.csv` dataset is located in the same directory as the notebook.
4. Launch Jupyter Notebook or JupyterLab and open `Countries.ipynb`.
5. Run the cells sequentially to reproduce the data transformations and visualizations.
